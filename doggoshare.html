Support

Need help with Doggo Share?

Doggo Share is designed to transfer files over your local network. To use it, open the app on your Mac, iPhone, or iPad, keep the app running, and open the displayed local address in a browser on another device connected to the same Wi-Fi network.

Basic Requirements

  • Make sure both devices are connected to the same Wi-Fi network.
  • Keep Doggo Share open while transferring files.
  • Do not lock the iPhone or iPad screen during transfer.
  • On macOS, choose a shared folder that you want to make available on the local network.
  • Use the browser address displayed in the app, such as http://192.168.x.x:7788.

Common Troubleshooting Tips

  • If the browser cannot open the address, check that both devices are on the same Wi-Fi network.
  • Disable VPN temporarily if the local address does not work.
  • Some guest Wi-Fi networks block communication between devices. Try using a normal private Wi-Fi network.
  • On iPhone or iPad, make sure local network permission is allowed in system settings.
  • On macOS, if the shared folder cannot be accessed, choose the folder again in the app.

Privacy Policy

Last updated: May 30, 2026

This Privacy Policy explains how Doggo Share handles information when you use the app.

1. Information We Collect

Doggo Share does not require account registration and does not collect your name, phone number, contacts, precise location, photo library content, or other personal identity information.

To provide its basic features, the app may use the following information locally on your device:

  • Your local network IP address, used to display the browser access address.
  • The shared folder path selected by you, used to read and save transferred files.
  • Your selected app language preference.
  • Your device name, used to show which device is being accessed from the browser page.

This information is used only for the app to function on your device. It is not uploaded to our servers.

2. File Transfer and Cloud Storage

Doggo Share does not provide cloud storage. Your files are not uploaded to the developer’s server and are not transferred through any external server.

File transfer happens only between your current device and another device on the same local network that opens the displayed local address in a browser.

3. Local Network Access

Doggo Share uses local network access to allow devices on the same Wi-Fi network to upload, download, refresh, and delete files in the selected shared folder.

Please use this app only on trusted networks. While the transfer service is running, devices that can access the displayed local address may be able to view or manage files in the shared folder. You can stop the transfer service at any time in the app.

4. Shared Folder Permissions

On macOS, you may choose a local folder as the shared folder. Doggo Share only accesses the folder you explicitly select. It does not scan other folders on your computer.

On iOS and iPadOS, the app uses its own app document folder as the shared folder. It does not access the full file system of your device.

5. Third-Party Services

Doggo Share’s core file transfer features do not rely on third-party cloud services. The app does not send your files to advertising platforms, analytics platforms, or external file storage servers.

6. Data Storage and Deletion

Files uploaded, downloaded, saved, or deleted through Doggo Share remain in the shared folder selected by you or in the app’s local document folder.

You may delete these files manually or through the browser file management page. Local settings, such as language preference and shared folder path, are stored on your device. These local settings are generally removed when the app is uninstalled.

7. Children’s Privacy

Doggo Share is not specifically designed for children. We do not knowingly collect personal information from children.

8. Changes to This Privacy Policy

We may update this Privacy Policy from time to time as the app changes. The updated version will be posted on this page.
